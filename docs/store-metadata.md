# Store Metadata — SOS Vidas (Nexus Automação)

## Nome do app
SOS Vidas (Nexus Automação)

## Descrição curta
Atalhos oficiais para primeiros socorros, números de emergência e unidades de atendimento próximas.

## Descrição longa
SOS Vidas (Nexus Automação) é um aplicativo mobile criado para facilitar o acesso rápido a orientações oficiais de primeiros socorros, números de emergência e localização de unidades de atendimento próximas.

O app foi estruturado para uso prático em situações de urgência, sem substituir atendimento profissional e sem emitir laudos médicos. Seu papel é organizar, dentro do próprio aplicativo, conteúdos oficiais e caminhos rápidos para ajuda.

### Recursos principais
- fichas de primeiros socorros organizadas por tema
- orientação clara entre 192 (SAMU) e 193 (Bombeiros)
- acesso rápido a números de emergência
- busca de hospitais, UPAs 24h e centros de saúde próximos
- navegação guiada para quem não sabe identificar a situação

### Observação importante
O app encaminha para conteúdo público/oficial e não substitui avaliação médica, diagnóstico ou tratamento profissional.
